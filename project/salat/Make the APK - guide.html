Turning the app build into an APK

The app is a self-contained web app. The fastest route to an installable APK is to wrap it with Capacitor — it puts the HTML inside a native Android WebView and produces a standard .apk. About 20 minutes from a clean machine.

What you have

Simple Salat — App (offline).html — one fully self-contained file (fonts, scripts, styles all inlined). It needs no internet and is what goes inside the app. Rename it index.html when you drop it in.

Prerequisites

Node.js 18+

For the Capacitor CLI. nodejs.org

Android Studio

Provides the Android SDK + Gradle that build the APK. developer.android.com/studio

Steps

  1. Create a project folder with a www/ subfolder. Copy the offline HTML into it as www/index.html. 
    mkdir simple-salat && cd simple-salat
mkdir www
# copy "Simple Salat — App (offline).html" → www/index.html
  2. Initialise Capacitor and add the Android platform. 
    npm init -y
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init "Simple Salat" com.simplesalat.app --web-dir=www
npx cap add android
  3. Sync the web files into the native project (re-run this any time the HTML changes). 
    npx cap sync
  4. Open in Android Studio and let it finish its first Gradle sync. 
    npx cap open android
  5. Build the APK. In Android Studio: Build → Build Bundle(s) / APK(s) → Build APK(s). The file lands at 
    android/app/build/outputs/apk/debug/app-debug.apk
    Copy it to the Kompakt and install (enable “install from unknown sources” if prompted).

Recommended polish before shipping

  1. Lock to portrait + fullscreen. In android/app/src/main/AndroidManifest.xml set the activity to android:screenOrientation="portrait"; for an immersive E Ink feel, hide the system bars.
  2. App icon. Replace the launcher icons (Android Studio → right-click res → New → Image Asset). A 1-bit version of the arc mark suits the device.
  3. Release signing. For a non-debug build, generate a keystore and use Build → Generate Signed Bundle / APK. Required if you ever distribute it.
