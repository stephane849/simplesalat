# Salāt

Prayer times for the [Mudita Kompakt](https://mudita.com/products/phones/mudita-kompakt/).

Designed for E-Ink — high-contrast, no animations, no unnecessary UI.

---

## Features

**Arc of the Day** — a sun-path arc from Fajr to Isha shows where you are in the prayer day. Tap any prayer name to highlight it and see its exact time and countdown.

**Background alarms** — the adhan plays at prayer time even when the phone is sleeping or the screen is locked. The screen wakes, the app opens over the lock screen, and a Stop button appears.

**Three bundled adhans** — Makkah, Madīnah, and Sudan recordings — with adjustable playback volume and in-app preview.

**Hijri calendar** — browse by month with a ±2-day adjustment for local moon sighting.

**Six calculation methods** — ISNA, Muslim World League, Egyptian Authority, Umm al-Qurā University, Karachi, Moonsighting Committee.

**Standard and Ḥanafī Asr** — select your school's shadow-length method.

**12h / 24h time display.** GPS location with manual fallback.

## Install

1. Download **salātapp.apk** from [Releases](../../releases/latest)
2. On the Kompakt: *Settings → System → Unknown sources* → enable
3. Open the APK in the file manager and tap Install

## Development

The app is plain React 18 (UMD, no bundler) wrapped by Capacitor 6. There is no local build step — pushing to `main` triggers a GitHub Actions workflow that compiles the JSX, generates the Android project, writes the custom Java plugins, and publishes a GitHub Release with the APK.

### Source layout

| Path | Contents |
|---|---|
| `www/app/` | JSX source files (compiled to `.js` by CI) |
| `www/css/` | Stylesheet |
| `www/audio/` | Bundled adhan MP3s |
| `www/index.html` | App entry point |
| `.github/workflows/build-apk.yml` | Full CI pipeline |

### Custom native plugins

| Plugin | Purpose |
|---|---|
| `AdhanSchedulerPlugin` | Schedules/cancels exact alarms via `AlarmManager` |
| `AdhanService` | Foreground service — plays adhan with `USAGE_ALARM`, holds wake lock, wakes screen |
| `SalatAudioPlugin` | In-app audio preview using native `MediaPlayer` |

All plugin Java is generated by the CI workflow — there is no `android/` directory in the repo.
